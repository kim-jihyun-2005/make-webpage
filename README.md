# make-webpage

# 웹 개발 응용 대체 과제 - 홈페이지 만들기

## 과제 목표
HTML, CSS, JavaScript를 활용해 영문 타자기, 한글 타자기, 계산기, 전화기의 기능을 구현하고,
하나의 홈페이지 형태로 제작함

## 구현한 기능 목록
- **초기 페이지 & 사이트맵** : `homepage.html`
- **영문 타자기** : `eng_typewriter.html`
- **한글 타자기** : `kor_typewriter.html`
- **계산기** : `calculator.html`
- **전화기** : `telephone.html`
- **GNB, Foooter** : 네비게이션 기능, 학번과 이름 표시
